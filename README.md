# WordsStudy

# About

The main purpose of this app to help to remember words easy by simple cards.
This app represents a set of a card which helps student to learn words.
Every word student would plan to repeat in next few minutes, next day or next few days.

Also this app allows add word to vocabulary by few ways:
* by hand typing
* by voice
* by photo camera
* by pressing on text for example in browser and then translate and add
